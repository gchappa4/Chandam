Chandam API  [Alpha]


Beautify :

It marks Yati and Prasa for the padyam that matches given criteria for chandassu.
  1. beautifyId("Id");   - For a DOM Element with given Id
  2. beautifyTags("p");   - For all given DOM Elements with given Tag
  3. beautifyAll();   - For a given DOM Element where attribute of chandam='1'
  4. beautify(elem);   - For a given DOM Element object.

Beautify :

It marks Yati and Prasa for the padyam that matches given criteria for chandassu.
  1. beautifyId("Id","chandassu");   - For a DOM Element with given Id
  2. beautifyTags("p","chandassu");   - For all given DOM Elements with given Tag
  3. beautifyAll();   - For a given DOM Element where attribute of chandam='1'

Beautify :

It marks Yati and Prasa for the padyam that matches given criteria for chandassu.
  1. beautifyId("Id","chandassu",{Additional Options});   - For a DOM Element with given Id
  2. beautifyTags("p","chandassu",{Additional Options});   - For all given DOM Elements with given Tag
  3. beautifyAll(null,null,{Additional Options});   - For a given DOM Element where attribute of chandam='1'
